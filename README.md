## 2 Инструкции за инсталация и стартиране

За стартиране на приложението са необходими Node.js (версия 18 или по-нова) и npm. Стъпките са следните:

### Стъпка 1 - Clone repo

Копирайте линка за клониране локално:

```bash
git clone <repo link>
```

### Стъпка 2 - Отворете терминал

Навигирайте до папката `task-manager`:

```bash
cd task-manager
```

### Стъпка 3 - Инсталиране на зависимостите

```bash
npm install
```

Тази команда изтегля React, React DOM, Vite и всички необходими пакети в папката `node_modules`.

### Стъпка 4 - Стартиране в режим за разработка

```bash
npm run dev
```

Vite стартира development сървър. В терминала ще се появи адрес от вида `http://localhost:5173`. Отворете го в браузъра и приложението е готово за използване.

### Стъпка 5 - Production build (по желание)

```bash
npm run build
```

Командата генерира оптимизирана production версия в папката `dist/`, подходяща за качване на уеб хостинг.

## 3 Забележки относно данните

Приложението съхранява задачите единствено в React state, тоест в паметта на браузъра. Данните се нулират при презареждане на страницата. Четири примерни задачи са предефинирани при инициализация на hook-а, за да демонстрират различните функционалности веднага при стартиране.

При желание за постоянно съхранение задачите лесно могат да бъдат персистирани в `localStorage` на браузъра чрез добавяне на `useEffect` в `useTaskManager.js`, без промени в останалите компоненти. Това демонстрира предимствата на разделянето на логиката от UI.
